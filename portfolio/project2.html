BANE

When a client needed to enable 2FA to their employee's accounts, they realized that not everyone had a smartphone to run an app. And in a few cases, some accounts were shared email boxes, meaning multiple people would have to coordinate all their apps in order to log in. To solve these issues, I developed BANE, a utility comprised of a locally installed user interface, hosted API server, and admin portal. I was able to design a secure method to identify specific computers and verify authenticated users before generating their 2FA code.

Users can be managed through the administrative web portal, and the local app was implemented in Tauri (a Rust-based electron alternative) which connects to a Dockerized-Python API server and SQLite database.
